また雑なプログラムを書いて公開しました。  
[https://github.com/ishkawa/ttw](https://github.com/ishkawa/ttw)

ターミナルにTwitterのUser Streamを流せます。
```
ttw stream
```

ツイートの投稿もできます。
```
ttw post hoge
```

Q. なんでつくったの？  
A. 起動するアプリを減らしたかったので。

Twitter Streaming APIを使ってみたい方には参考になるかもしれません。
