手元ではCoverStoryを使っていたのですが、なんか使いづらかったのでgcovrを使い始めました。  
gcovr 3.1にはhtml出力がついていて、以下のようなコマンドでレポートを出力してくれます。

```
gcovr \
	-r . \
	--html --html-detail \
	-o build/coverage.html \
	-e Pods \
	-e UnitTests \
	-e ".*\.h"
```
